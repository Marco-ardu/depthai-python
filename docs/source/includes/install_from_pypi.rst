Please run the `install script <https://gitee.com/oakchina/depthai-python/blob/main/examples/install_requirements.py>`__
to download all required dependencies. Please note that this script must be ran from git context, so you have to download the `depthai-python <https://gitee.com/oakchina/depthai-python>`__ repository first and then run the script

.. code-block:: python
   :substitutions:

   git clone https://gitee.com/oakchina/depthai-python.git
   cd depthai-python/examples
   python3 install_requirements.py


For additional information, please follow `installation guide <https://docs.oakchina.cn/en/latest/pages/startQuickly/Python.html>`__
